# 信号筛选规则手册

> 生成日期：2026-05-19
> 数据基础：`stocks/ml/shadow_data.json` 当前共 2327 条已回填样本
> 目标：从金叉信号中筛选出未来 5 个交易日内更可能触达目标价、并具备较高弹性的股票。
> 说明：本手册整合了原规则手册结论，并基于当前最新 `shadow_data.json` 重新验证。由于样本持续回填，部分指标与旧版 1927 条样本版本不同。

---

## 一、核心结论

当前最新数据重新验证后，规则效果如下：

| 规则版本 | 条件数 | 样本数 | 达标数 | 达标率 | 高弹性率≥8% | 平均最大涨幅 | 平均实际收益 |
|---|---:|---:|---:|---:|---:|---:|---:|
| 无筛选 | 0 | 2327 | 439 | 18.9% | 20.8% | 4.6% | -0.0% |
| V1 基础规则 | 4 | 117 | 28 | 23.9% | 29.9% | 6.3% | +0.02% |
| V2 标准规则 | 7 | 43 | 17 | 39.5% | 44.2% | 8.7% | +0.04% |
| V2 + 主力 2000~4000万 | 7+ | 20 | 11 | 55.0% | 55.0% | 11.5% | +0.07% |
| V3 强信号规则 | 7+ | 9 | 6 | 66.7% | 66.7% | 15.9% | +0.12% |

### 结论概括

1. **V1 四条基础规则有效，但提升有限。**
   从无筛选 18.9% 提升到 23.9%，主要作用是排除明显低质量信号。

2. **V2 三条新增规则有效。**
   加入“涨幅确认、大单过滤、日线周期”后，达标率提升到 39.5%，高弹性率提升到 44.2%。

3. **当前数据不支持继续使用旧版手册中的 60.7% 作为 V2 稳定胜率。**
   旧版手册基于 1927 条样本，当前数据为 2327 条，重新统计后 V2 为 43 条、达标 17 条，达标率 39.5%。

4. **资金流入不是越多越好。**
   当前数据中，V2 样本内主力净流入 2000~4000 万明显优于 4000 万以上。

5. **资金流入占比上限建议从 20% 收紧到 12%。**
   V2 样本内 1~12% 达标率 48.5%，而 12~20% 仅 18.2%。

6. **大单净流入 <4000 万方向正确，强信号可进一步收紧到 <2000 万。**
   V2 样本内大单 2000~4000 万表现较差，达标率仅 23.1%。

## 二、几个关键指标的含义

| 字段 | 中文含义 | 说明 |
|---|---|---|
| `sc_close` | 信号日收盘价 | 买入参考价 |
| `an_capital_main_net_in` | 主力净流入，单位万元 | 超大单 + 大单的净流入金额 |
| `an_capital_flow_ratio` | 主力流入强度，占比% | 主力净流入 / 当日成交额 × 100%。对应钉钉推送里“净买入 +xxxx万(+x.x%)”括号内的数值 |
| `an_success_rate_dim_momentum` | 动能评分 | 趋势、MACD、均线、量价等综合动能评分 |
| `an_quote_change_pct` | 信号日涨跌幅% | 信号当天市场是否真正发力 |
| `an_capital_big_net_in` | 大单净流入，单位万元 | 大单资金净流入，过高时可能存在拉高出货或突击资金嫌疑 |
| `period` | 信号周期 | 日线 / 30分钟 / 5分钟 |
| `reached_target` | 是否达标 | 未来 5 个交易日内最高价是否触达目标价 |
| `max_gain_pct` | 最大涨幅 | 未来 5 个交易日内相对信号日收盘价的最大涨幅 |
| `actual_return` | 实际收益 | 信号日收盘价买入，第 5 个交易日收盘价卖出的收益 |

## 三、V1 基础规则

V1 是原始规则，主要用于排除低质量金叉信号。

```python
def v1_filter(record):
    return (
        record.get('sc_close', 0) >= 10 and
        record.get('an_capital_main_net_in', 0) >= 2000 and
        1 <= record.get('an_capital_flow_ratio', 0) <= 20 and
        record.get('an_success_rate_dim_momentum', 0) >= 95
    )
```

### V1 当前效果

| 规则 | 样本数 | 达标率 | 高弹性率 | 平均最大涨幅 | 平均实际收益 |
|---|---:|---:|---:|---:|---:|
| 无筛选 | 2327 | 18.9% | 20.8% | 4.6% | -0.0% |
| V1 | 117 | 23.9% | 29.9% | 6.3% | +0.02% |

### V1 四条规则评价

| 条件 | 评价 |
|---|---|
| 股价 ≥10元 | 合理。用于过滤低价股噪音，但单独提升不大。后续建议加上限，10~60 元优于单纯 ≥10。 |
| 主力净流入 ≥2000万 | 合理。确认有资金推动。但不能越多越好，建议强信号加上 4000 万或 5000 万上限。 |
| 资金占比 1~20% | 方向正确，但上限偏宽。当前数据中 12% 以上明显变差。 |
| 动能 ≥95 | 合理。是基础趋势确认条件。 |

## 四、V2 标准规则

V2 在 V1 基础上新增三条：

1. 信号日涨幅 ≥3%
2. 大单净流入 <4000 万
3. 周期 = 日线

```python
def v2_filter(record):
    return (
        record.get('sc_close', 0) >= 10 and
        record.get('an_capital_main_net_in', 0) >= 2000 and
        1 <= record.get('an_capital_flow_ratio', 0) <= 20 and
        record.get('an_success_rate_dim_momentum', 0) >= 95 and
        record.get('an_quote_change_pct', 0) >= 3 and
        record.get('an_capital_big_net_in', 0) < 4000 and
        record.get('period') == '日线'
    )
```

### V2 当前效果

| 规则 | 样本数 | 达标数 | 达标率 | 高弹性率 | 平均最大涨幅 | 平均实际收益 |
|---|---:|---:|---:|---:|---:|---:|
| V1 | 117 | 28 | 23.9% | 29.9% | 6.3% | +0.02% |
| V2 | 43 | 17 | 39.5% | 44.2% | 8.7% | +0.04% |

### V2 三条新增规则的贡献

在 V1 基础上逐项叠加验证：

| 规则组合 | 样本数 | 达标率 | 高弹性率 | 平均最大涨幅 | 平均实际收益 |
|---|---:|---:|---:|---:|---:|
| V1 基础 | 117 | 23.9% | 29.9% | 6.3% | +0.02% |
| V1 + 涨幅≥3% | 83 | 30.1% | 33.7% | 7.1% | +0.03% |
| V1 + 大单<4000万 | 85 | 28.2% | 36.5% | 7.2% | +0.02% |
| V1 + 日线 | 85 | 25.9% | 30.6% | 6.5% | +0.02% |
| V1 + 涨幅≥3% + 大单<4000万 | 59 | 35.6% | 42.4% | 8.5% | +0.04% |
| V1 + 涨幅≥3% + 日线 | 60 | 33.3% | 35.0% | 7.3% | +0.03% |
| V1 + 大单<4000万 + 日线 | 63 | 30.2% | 38.1% | 7.4% | +0.03% |
| V2 三条全加 | 43 | 39.5% | 44.2% | 8.7% | +0.04% |

### 数据结论

- `an_quote_change_pct >= 3` 是新增条件中最明显的动能确认条件。
- `an_capital_big_net_in < 4000` 有效提升高弹性率。
- `period == 日线` 单独提升不算大，但和涨幅、大单过滤组合后效果更好。

## 八、实盘使用提醒

### 1. 不要把规则当成自动买入系统

规则只能帮助缩小观察范围。真正买入前还需要看：

- 是否连续大涨后追高；
- 是否一字板无法参与；
- 是否冲高回落；
- 是否处于板块主线；
- 是否有明显利空或退市风险；
- 大盘环境是否支持短线行情。

### 2. 资金类指标要反着警惕

以下情况不是加分，而是风险：

```text
主力净流入特别巨大
资金占比超过15%
大单净流入超过4000万
```

这类信号可能是拉高出货、短期突击资金、或者数据结构异常。

### 3. 样本数越少，越不能迷信达标率

例如 V3 强信号当前达标率 66.7%，但只有 9 条样本。

正确理解应为：

```text
它是一个值得重点关注的强信号标签，
不是一个已经被充分验证的高胜率交易系统。
```

### 4. 建议持续滚动验证

每新增一批回填样本后，应重新统计：

- V1 / V2 / V2改良 / V3 的样本数；
- 达标率；
- 高弹性率；
- 平均最大涨幅；
- 平均实际收益；
- 被过滤掉但实际大涨的误杀样本。

## 九、最终推荐

当前最建议使用的规则不是旧 V1，也不是过拟合风险较高的 V3，而是：

```text
主规则：V2 改良版
股价 >=10
主力净流入 >=2000万
资金占比 1~12%
动能 >=95
信号日涨幅 >=3%
大单净流入 <4000万
周期 = 日线
```

然后额外打强信号标签：

```text
强信号标签：
10 <= 股价 <60
2000万 <= 主力净流入 <=4000万
资金占比 1~12%
大单净流入 <2000万
其余条件同 V2
```

一句话总结：

> V1 是基础过滤，V2 是有效升级，当前最值得采用的是“V2 改良版”：保留涨幅确认、大单过滤、日线周期，同时把资金占比从 1~20% 收紧到 1~12%。强信号可以进一步要求主力净流入 2000~4000 万、大单净流入 <2000 万，但由于样本较少，只能作为重点关注标签。
