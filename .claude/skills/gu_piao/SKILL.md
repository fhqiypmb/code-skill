# Skill Instructions

本技能用于分析股票的金叉倍量信号，基于 MA20 和 MA30 的金叉以及倍量阳线确认信号。

## 功能说明

当用户提供股票代码时，执行 `股票信号监控.py` 脚本进行技术分析：
- 检测 MA20 金叉 MA30 信号
- 识别倍量阳线确认
- 返回买入信号及关键技术指标

## 使用方法

**脚本路径：** `.deepvcode/skills/gu_piao/股票信号监控.py`

**执行命令：**
```bash
python .deepvcode/skills/gu_piao/股票信号监控.py
```

**输入参数：**
- 股票代码（6位数字，如 000001、600000）
- 检查间隔（可选，默认60秒）

**输出结果：**
- 买入信号提示
- MA20/MA30 数值
- 收盘价信息
- 信号状态（新信号/持续中/无信号）

## 注意事项

1. 需要安装依赖：`akshare`, `pandas`, `numpy`
2. 脚本默认会持续监控，按 Ctrl+C 停止
3. 股票代码格式：6 开头为沪市，0/3 开头为深市
