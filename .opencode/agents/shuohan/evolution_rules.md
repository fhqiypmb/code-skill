# System.md 自进化机制 v2.0

## 触发条件（满足任一即触发）

1. **Agent 记忆积累**：最近 7 天新增 5 条以上经验
2. **用户偏好变化**：最近 7 天新增 3 条以上偏好
3. **用户明确要求**：用户要求修改系统行为
4. **定期评估**：每 50 次对话后自动评估

## 自动检查机制

Agent 在每次启动时会自动检查是否满足进化条件：

```bash
python .opencode/skills/shuohan/system_evolve.py
```

如果满足条件，会提示用户是否执行进化。

## 进化流程

1. **分析触发原因**：统计 agent.md 和 cold.md 的最近新增
2. **备份当前版本**：保存到 backup 目录，文件名包含版本号和时间戳
3. **LLM 智能分析**：
   - 读取当前 system.md
   - 读取最近的 agent 和 cold 记忆
   - 提取可固化的经验和偏好
   - 生成新版本的 system.md
4. **写入新版本**：更新 system.md，版本号递增
5. **记录进化日志**：在 agent.md 中记录本次进化的原因和内容

## 进化原则

### 从 Agent 记忆中提取：

- 可复用的工作流程
- 技术规范和最佳实践
- Bug 修复经验
- 架构决策

### 从 Cold 记忆中提取：

- 用户的默认行为偏好
- 安全规则和确认机制
- 工作习惯和风格

### 保持不变：

- 核心身份和定位
- 基础工作流程
- 记忆系统架构

## 安全机制

- **备份保护**：每次进化前自动备份
- **版本管理**：保留最近 5 个历史版本
- **用户确认**：重大变更需用户确认
- **回滚能力**：可随时恢复到历史版本

## 回滚方法

如果进化后出现问题，可以回滚到之前的版本：

```bash
# 查看备份版本
ls -lt .opencode/agents/shuohan/backup/

# 恢复指定版本
cp .opencode/agents/shuohan/backup/system_v7.0_20260302_120000.md .opencode/agents/shuohan/prompts/system.md
```

## 进化示例

### 示例 1：固化安全规则

**触发原因**：Cold 记忆中记录了 3 条关于"操作前确认"的偏好

**进化内容**：在 system.md 中添加"安全操作清单"章节，明确哪些操作需要用户确认

### 示例 2：固化技术经验

**触发原因**：Agent 记忆中记录了 5 条关于"编码问题"的修复经验

**进化内容**：在 system.md 中添加"编码处理规范"，要求所有文件操作强制使用 UTF-8

### 示例 3：优化工作流程

**触发原因**：Agent 记忆中记录了多次"忘记写入记忆"的问题

**进化内容**：强化 system.md 中的"强制写入"规则，增加检查点
