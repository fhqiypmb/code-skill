# 🚀 技能使用超简单教程

## 第一步：了解您现在有哪些技能

**您只需要对我说：**
```
请帮我列出所有可用的技能
```

**我会做什么：**
- 显示所有技能的列表
- 包括技能名称和简短描述

---

## 第二步：查看某个技能的详细信息

**您只需要对我说：**
```
请帮我查看 my_first_skill 的详细信息
```

**我会做什么：**
- 读取那个技能的 SKILL.md 文件
- 把完整内容展示给您

---

## 第三步：激活并使用技能

**您只需要对我说：**
```
请使用 my_first_skill 技能
```

**我会做什么：**
- 尝试激活这个技能
- 如果是**知识型技能**（没有脚本）：我会记住它的内容，然后按照它的指导来回答问题
- 如果是**脚本型技能**（有脚本）：我会告诉您需要运行什么命令

---

## 💡 关键理解

### 什么是"激活技能"？

**简单来说：**
- 激活技能 = 让我读取那个技能的 SKILL.md 文件
- 读取后 = 我知道了那个技能想让我做什么
- 然后 = 我按照技能的指导来工作

### 两种技能的区别

**知识型技能（简单）：**
- 只有 SKILL.md 文件
- 激活后，我的行为会改变（比如扮演某个角色，或按特定方式回答）
- 不需要运行任何命令

**脚本型技能（复杂）：**
- 有 SKILL.md + Python/JavaScript 脚本
- 激活后，我会告诉您需要运行什么命令
- 您需要让我执行那些命令来完成任务

## 🔧 如果遇到问题

### 问题：技能列表里没有我创建的技能
**原因：** 系统可能需要刷新
**解决：** 重启 VS Code 或者让我直接读取文件

### 问题：激活技能失败
**原因：** 技能加载系统可能有缓存
**解决：** 让我直接读取 SKILL.md 文件，效果是一样的
