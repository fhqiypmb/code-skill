# 我的第一个技能 🎉

## 这个技能是什么？

这是一个**演示技能**，专门用来教您如何使用技能系统。

## 激活这个技能后，AI 会做什么？

当您说"请使用 my_first_skill 技能"时，AI 会：
1. 读取这个文档
2. 记住这个技能的内容
3. 按照这个技能的指导来回答您的问题

## 这个技能提供什么功能？

### 功能 1️⃣：打招呼
当您激活这个技能后，AI 会用友好的方式向您打招呼，并告诉您技能已经激活。

### 功能 2️⃣：回答问题
这个技能让 AI 扮演一个"技能系统导师"的角色，专门回答关于技能的问题：
- 技能是什么？
- 如何创建技能？
- 如何使用技能？

## 使用示例

激活技能后，您可以这样问：
- "技能到底是什么？"
- "我怎么创建自己的技能？"
- "给我讲讲技能系统的原理"

AI 会以简单易懂的方式回答您。

## 如何激活这个技能？

### 方法 1：直接说（推荐）
直接对我说：
```
请使用 my_first_skill 技能
```

### 方法 2：让我先查看
您也可以先让我查看技能详情：
```
请帮我查看 my_first_skill 的详细信息
```
然后再说：
```
好的，请使用这个技能
```

## 重要提示 ⚠️

- 这是一个**知识型技能**（没有脚本）
- 激活后不需要运行任何命令
- 它只是改变了 AI 的回答方式和角色定位
