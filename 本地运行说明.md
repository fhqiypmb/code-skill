# .bat 使用说明

项目根目录有 4 个 .bat：

| 文件 | 干啥 |
|------|------|
| `本地启动.bat` | **日常用这个**，双击弹菜单选 |
| `自动启动.bat` | 任务计划自动调用（静默跑选股+盯盘），不用手点 |
| `周训练.bat` | 任务计划自动调用（ML训练），不用手点 |
| `配置自动运行.bat` | 想自动跑时双击一次，注册任务计划 |

---

## 日常：双击 `本地启动.bat`

菜单：
```
1. 选股监控
2. 持仓盯盘
3. 选股+盯盘 同时跑   ← 日常用这个
4. 选股监控 --now    (立即测试)
5. 持仓盯盘 --now    (立即测试)
6. ML 周训练         (每周一跑一次)
7. 更新股票列表
0. 退出
```

- 交易日早上：选 `3`，挂到收盘自动退出
- 每周一：选 `6`，训练一次

---

## 想自动跑：双击 `配置自动运行.bat`（一次即可）

注册后每工作日 08:50 自动启动（脚本内部等到 09:40 开盘才开扫）、周一 08:00 自动训练，不用再手动点。
（电脑要开机；如报权限失败，右键“以管理员身份运行”）

**改/取消**：`Win+R` 输入 `taskschd.msc`，找到任务 `股票_选股盯盘`、`股票_ML周训练` 修改或删除。
